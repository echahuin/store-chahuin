## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
Don't forget to run `yarn install`
Use node version: `18.17.0`

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
